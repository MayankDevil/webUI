Bodhi Linux

The Enlightened Linux Distribution

Bodhi Linux is a lightweight distribution based on Ubuntu, designed to provide a simple and efficient computing experience. It features the Moksha Desktop Environment, a fork of the Enlightenment desktop that emphasizes minimalism and speed. Bodhi Linux is an excellent choice for those seeking a minimal yet functional Linux distribution. Its combination of a lightweight design, customizable desktop environment, and easy file management makes it appealing for new and experienced users alike.

Latest Version 7.0.0 →

Bodhi Linux is an open-source distro with a rather small team of active Team Members; we are always looking for help. In particular, we are looking for C, Bash, and Python programmers, translators, graphic artists, theme developers, and more. If you have skills to put to use, or a willingness to learn, contact us via Discord or our forums. Not all users can nor want to contribute to the code or documentation, but even a small financial donation — whether through PayPal or Patreon — helps tremendously.
